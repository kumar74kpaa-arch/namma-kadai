# **App Name**: Namma Kadai

## Core Features:

- Product Browsing: Display all available products with images, names, and prices.
- Product Details: Show detailed information about each product, including image, description, and 'Add to Cart' button.
- Shopping Cart: Display added items and total price; allow checkout with customer name, phone, and address.
- Admin Authentication: Secure login for admin using Firebase Authentication to manage products.
- Product Management: Admin interface to add, edit, or delete products with photo, name, price, and description. Saving to Firestore, photos to Storage.
- Image Storage: Upload product images to Firebase Storage and store the image URL in Firestore with product info.
- Tamil Text Support: Ensure proper display of Tamil text across the entire UI.
- Footer: Display 'Made by Kumar Technologies, kumartech.unaux.com and designed by S Sarika' in the app footer.

## Style Guidelines:

- Primary color: Golden Yellow (#FFC107), representing richness and festivity, common in Tamil culture.
- Background color: Beige (#F5F5DC), evoking the color of undyed cotton and banana leaves; conveys warmth and simplicity.
- Accent color: Brown (#A67B5B), drawing inspiration from wooden textures for an earthy feel.
- Font: 'Literata', serif font that brings literary/traditional touch
- Use icons inspired by Tamil culture, such as traditional lamps (kuthu vilakku), kolam patterns, or mango leaves.
- Responsive layout suitable for both Android and iOS devices, optimized for different screen sizes.
- Smooth splash screen animation displaying coconut trees, namaste symbol, and Tamil text 'வணக்கம்'.
